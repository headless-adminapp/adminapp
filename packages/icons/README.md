# @headless-adminapp/icons

Shared icon set for Headless AdminApp.

## Overview

This package provides the base icon components used across Headless AdminApp packages and UI kits.

## Installation

```sh
pnpm add @headless-adminapp/icons
```

## Usage

Import icons from the package entry point according to the documentation:

- https://headless-adminapp.github.io/

## Development

```sh
pnpm --filter @headless-adminapp/icons build
pnpm --filter @headless-adminapp/icons ts-check
```
