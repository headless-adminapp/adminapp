# @headless-adminapp/server-sdk-sequelize

Sequelize adapter for the Headless AdminApp server SDK.

## Overview

This package provides a Sequelize-based storage adapter for Headless AdminApp server-side integrations.

## Installation

```sh
pnpm add @headless-adminapp/server-sdk-sequelize
```

Peer dependencies:

- sequelize

## Usage

Use this package alongside @headless-adminapp/server-sdk. See the docs and examples for detailed setup:

- https://headless-adminapp.github.io/
- https://github.com/kishanmundha/headless-adminapp-examples

## Development

```sh
pnpm --filter @headless-adminapp/server-sdk-sequelize build
pnpm --filter @headless-adminapp/server-sdk-sequelize test
```
