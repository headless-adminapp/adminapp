# @headless-adminapp/next

Next.js integration helpers for Headless AdminApp.

## Overview

This package provides utilities for using Headless AdminApp in Next.js applications, including helpers for routing and response formatting.

## Installation

```sh
pnpm add @headless-adminapp/next
```

Peer dependencies:

- next

## Usage

See the documentation and example apps for recommended Next.js setup:

- https://headless-adminapp.github.io/
- https://headless-adminapp-examples.vercel.app/

## Development

```sh
pnpm --filter @headless-adminapp/next build
pnpm --filter @headless-adminapp/next ts-check
```
