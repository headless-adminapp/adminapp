# @headless-adminapp/core

Framework-agnostic core runtime and types for Headless AdminApp.

## Overview

This package contains the core building blocks used across the Headless AdminApp ecosystem. It is intended to be UI-agnostic so it can power multiple UI implementations.

## Installation

```sh
pnpm add @headless-adminapp/core
```

Peer dependencies:

- @tanstack/react-query

## Usage

Start with the project documentation and example apps for recommended integration patterns:

- https://headless-adminapp.github.io/
- https://headless-adminapp-examples.vercel.app/

## Development

```sh
pnpm --filter @headless-adminapp/core build
pnpm --filter @headless-adminapp/core ts-check
```
