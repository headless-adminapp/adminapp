# @headless-adminapp/icons-fluent

Fluent UI icon adapters for Headless AdminApp.

## Overview

This package bridges the Headless AdminApp icon abstractions to Fluent UI icons. It is intended for use with the Fluent UI implementation layer.

## Installation

```sh
pnpm add @headless-adminapp/icons-fluent
```

Peer dependencies:

- @fluentui/react-icons

## Usage

Use this package alongside @headless-adminapp/fluent and follow the project docs for integration details:

- https://headless-adminapp.github.io/

## Development

```sh
pnpm --filter @headless-adminapp/icons-fluent build
pnpm --filter @headless-adminapp/icons-fluent ts-check
```
