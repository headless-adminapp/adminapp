# @headless-adminapp/server-sdk-mongo

MongoDB adapter for the Headless AdminApp server SDK.

## Overview

This package provides a MongoDB/Mongoose storage adapter for Headless AdminApp server-side integrations.

## Installation

```sh
pnpm add @headless-adminapp/server-sdk-mongo
```

Peer dependencies:

- mongoose

## Usage

Use this package alongside @headless-adminapp/server-sdk. See the docs and examples for detailed setup:

- https://headless-adminapp.github.io/
- https://headless-adminapp-examples.vercel.app/

## Development

```sh
pnpm --filter @headless-adminapp/server-sdk-mongo build
pnpm --filter @headless-adminapp/server-sdk-mongo test
```
